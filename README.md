# Chantaro Mobile

Mobile shop landing page built with HTML, CSS and Bootstrap.  
HTML・CSS・Bootstrap を使用したモバイルショップ紹介サイトです。

---

## Overview | プロジェクト概要

Chantaro Mobile is a responsive landing page project for practicing frontend development and UI design.

Chantaro Mobile は、フロントエンド開発とUIデザイン学習のために制作したレスポンシブWebサイトです。

---

## Tech Stack | 使用技術

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- AOS Animation

---

## Features | 主な機能

- Responsive Design
- Mobile Friendly UI
- Product Showcase Section
- Smooth Scroll Animation
- Bootstrap Layout System
